# AI 工作台 Demo ASCII Wireframe v2

基于 `other/WorkbenchAscii.md`、`other/WorkbenchWireframes.md`、`other/WorkbenchDemo.md`，这版 v2 目标只有两个：
- 统一 5 个页面的 Top Bar / Left Nav / Right Panel 排版风格，让它们看起来属于同一套系统
- 给每个页面补齐模块级字段说明，方便直接进入 Demo 页面开发

统一约定：
- 默认角色：`区域运营负责人`
- 默认范围：`华东区 · 182 家门店`
- 统一三栏：`Left Nav / Main Workspace / Right Panel`
- 右侧 Tab 顺序固定：`[任务] [提醒] [报告]`
- 页面默认高亮：
  - 首页：`提醒`
  - 经营分析页：`报告`
  - 门店风险页：`任务`
  - 任务协同页：`任务`
  - 报告中心页：`报告`

统一全局结构：
- `Top Bar`: 产品名、角色/范围、日期、搜索、通知、用户入口
- `Left Nav`: 模式切换、主导航、最近会话、快捷提问
- `Right Panel`: 任务/提醒/报告的轻量沉淀区，不承载主分析内容

### 模块级字段说明

- `Top Bar`
  - 字段：`产品名`、`角色`、`管理范围`、`日期`、`全局搜索`、`通知`、`用户入口`
  - 状态：固定展示当前角色和范围
  - 按钮：`日期切换`、`搜索`、`通知`

- `Left Nav`
  - 字段：`模式切换`、`主导航`、`最近会话`、`快捷提问`
  - 状态：当前页高亮 `今日工作台`
  - 按钮：模式切换按钮、导航项、会话项、快捷提问按钮

- `Summary Strip`
  - 字段：`高风险门店数`、`待处理任务数`、`新报告数`、`命中提醒数`
  - 状态：数值型摘要卡
  - 按钮：点击卡片跳转对应页面或筛选结果

- `Hero Card`
  - 字段：`结论标题`、`3 条今日重点结论`
  - 状态：用于高亮今日优先事项
  - 按钮：`[为什么利润下降]`、`[查看异常门店]`、`[生成整改任务]`

- `Recommended Questions`
  - 字段：预设问题文案
  - 状态：无
  - 按钮：每个问题可直接发起对话

- `Default Conversation Thread`
  - 字段：`用户问题`、`AI 结论`、`归因列表`
  - 状态：需明确区分 `确定性数据` 与 `AI 建议`
  - 按钮：`[展开看南京门店]`、`[只看外卖渠道]`、`[生成运营整改任务]`

- `Right Panel - 提醒`
  - 字段：`提醒标题`、`命中规则`、`受影响门店数`、`建议动作`
  - 状态：默认展示 3 条，更多进入列表页
  - 按钮：`[查看全部提醒]`

### 模块级字段说明

- `Top Bar`
  - 字段：`产品名`、`角色/范围`、`日期`、`搜索`、`通知`
  - 状态：固定
  - 按钮：日期切换、搜索、通知

- `Left Nav`
  - 字段：模式切换、主导航、最近会话、快捷提问
  - 状态：当前页高亮 `经营分析`
  - 按钮：导航项、会话项、快捷提问按钮

- `Query Bar`
  - 字段：`当前问题文本`
  - 状态：可编辑
  - 按钮：`[修改问题]`、`[保存为报告]`、`[生成任务]`

- `Conclusion Card`
  - 字段：`一句话结论`、`影响范围`、`优先动作`、`标签`
  - 状态：必须区分 `确定性数据` 和 `AI 建议`
  - 按钮：可复用 `[保存为报告]`

- `Insight Cards`
  - 字段：`归因标题`、`指标变化值`、`说明`
  - 状态：高亮异常幅度
  - 按钮：后续可扩展成点击展开详情

- `Visualization Area`
  - 字段：`趋势图`、`热力图`
  - 状态：按当前 query 和时间范围刷新
  - 按钮：后续可补充维度切换

- `Detail Table`
  - 字段：`门店`、`利润率`、`包装成本`、`补贴金额`、`差评率`、`风险等级`
  - 状态：支持排序，高风险行高亮
  - 按钮：点击行进入门店明细或生成任务

- `Follow-up Questions`
  - 字段：预设追问
  - 状态：无
  - 按钮：每个追问触发一次新的分析上下文

- `Right Panel - 报告`
  - 字段：`报告标题`、`状态`、`下次推送时间`
  - 状态：默认高亮报告
  - 按钮：`[保存当前分析为报告]`、`[查看全部报告]`

- `Top Bar`

  - 字段：`高风险`、`中风险`、`今日新增`、`已闭环`
